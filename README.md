

## 📊 Sales Analysis Dashboard (Excel)

### 🚀 Overview

This project presents an interactive **Sales Analysis Dashboard** built using Microsoft Excel.
It provides key insights into sales performance, customer distribution, and profitability across different categories and states.

---

### 📌 Key Features

* 💰 **Total Sales:** $105,887
* 📈 **Total Profit:** $726.92
* 📦 **Total Quantity Sold:** 3,127
* 📊 **Profit Margin:** 15%

---

### 📊 Dashboard Insights

* Sales distribution across **categories** (Furniture, Office Supplies, Technology)
* Profit analysis by **category and subcategory**
* Identification of:

  * 🔝 Top 10 products by sales
  * 🔻 Bottom 10 products by sales
* 📅 Performance trends over time
* 🗺️ Geographic analysis of **customers by state**

---
